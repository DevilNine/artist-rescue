**Artist Rescue** is a premium, specialized desktop application designed to solve the biggest headaches digital artists face: lost workspaces, corrupted brushes, driver crashes, and broken software updates. 

Whether you use Clip Studio Paint, MediBang, or FireAlpaca, Artist Rescue automatically secures your entire creative environment and keeps your tools running perfectly.

---

## ✨ Features

- **One-Click Backup & Restore**
  Instantly back up brushes, shortcuts, workspaces, and preferences for your favorite art software. Never lose your creative setup when formatting your PC or moving to a new one.

- **Tamper-Proof Archives**
  Every backup is cryptographically sealed using SHA-256 HMAC signatures. This ensures your backups are never corrupted, tampered with, or broken when you need them most.

- **Supported Software**
  - Clip Studio Paint
  - MediBang Paint Pro
  - FireAlpaca
  - JUMP PAINT
  - VRoid Studio
  - VTube Studio

- **Smart Software Management**
  Missing a program? Artist Rescue features an integrated installer for free apps (MediBang, FireAlpaca, JUMP PAINT) using the official Windows Package Manager (Winget), ensuring you always get the clean, official release.

- **Tablet Driver First-Aid**
  Pen pressure stopped working? The built-in driver fix tool instantly restarts Wacom, Huion, and generic tablet services without needing to reboot your computer.

- **System Dependency Checker**
  Automatically scans and installs critical system components (Visual C++ Redistributables, WebView2, .NET Framework) that art programs need to run without crashing.

## 🚀 Download & Installation

You can download the latest version from the **[Releases](../../tree/master/releases)** folder.

1. Choose either the **Setup Installer** (recommended) or the **Portable version**.
2. Run the application.
3. Your installed art software will be automatically detected!

## 🛡️ Security First
Artist Rescue is built with strict security standards. Backups are validated against path traversal attacks and ZIP structure corruption before being restored, ensuring your system remains safe and stable.
